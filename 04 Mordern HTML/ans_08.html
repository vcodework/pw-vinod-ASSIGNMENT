What is HTML

HTML, or Hypertext Markup Language, is the standard language used to create web pages. It uses tags to define the structure and content of a webpage, allowing for the inclusion of text, images, links, and other elements.

What are HTMl semantic?

HTML semantics refers to the proper use of HTML elements to give meaning and structure to web content. By using elements like header, nav, section, and footer, we can create a clear and well-organized document structure that is both readable by humans and understood by search engines.

Lists of common semantics tags

  • header: Represents the introductory content of a page or a section.
  • nav: Defines a set of navigation links.
  • main: Represents the main content of the page, and there should be only one
    element per page.
  • article: Represents a self-contained, independent piece of content, such as a blog post, news article, or comment.
  • section: Represents a standalone section of content that is thematically related.
  • aside: Represents content that is tangentially related to the main content, such as sidebars, pull quotes, or advertising.
  • footer: Represents the footer of a section or the whole page, containing information such as copyright notice, contact details, or related links.

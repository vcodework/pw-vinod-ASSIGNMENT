Comments

Comments are some text or code written in your code to give an explanation about the code, and not visible to the user.Comments which are used for HTML file are known as HTML comments. Anything written between these tags will be ignored by the browser, so comments will not be visible on the webpage. Comments of any code make code easy to understand and increase readability of code. Comments are also part of the code, which gives an explanation of the code.In HTML code, we can also comments multiple lines at a time. In multiline comment we can use any description about code or multiple line code to debug, etc.
